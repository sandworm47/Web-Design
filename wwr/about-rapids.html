Learn about the rapids!

Stay safe out there

While riding the white water can be a fantastic and thrilling experience, mother nature commands her rivers with a power we must respect. Learn about the different classes of rapids, and which ones work for you and your skills.

Class 1

Class 1 rapids are the perfect start for those new to water activities. Class 1 is classified as a smooth, yet swift river.

Class 2

Class 2 rapids are faster and rougher than class 1, and a perfect start for those new to rafting. Everything is dangerous no matter the class. Just because Class 2 is still a lower catgory does not mean it is harmless. A life jacket is strongly reccomended at this point, as well as a helmet.

Class 3

Class 3 is where the fun begins! While still managable for beginners, it is necessary for you to have an experienced guide to lead you through the gauntlet. It is also required for you to be able to swim at this stage.

Class 4

Class 4 can be dangerous if you don't know what to do. At this point, only experienced rafters should attempt a class 4.

Rapid classes go from 1 all the way to 6, but our services only take you as high as 4. Please visit another webiste, such as the one below, for more adrenaline pumping action!
